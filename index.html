HTML Table Cheatsheet

Tag Name Description
<table> table Wrapper element for all HTML tables
<thead> table head Set of rows defining column headers
<tr> table row element Adds rows to a table before adding table data and headings
<th> table heading element Adds titles to rows and columns of a table and must be enclosed in a table row element
<tbody> table body element Semantic element that contains all table data other than table heading and table footer content
<td> table data element Adds a cell of data to a table
<tfoot> table foot element Set of rows defining a footer in a table

Attribute Description
colspan Indicates how many columns that a particular cell should span within a table
rowspan Indicates how many rows a particular cell should span within a table
